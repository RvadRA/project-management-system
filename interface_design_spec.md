# Проектирование интерфейса - Система управления проектами

## 1. Назначение системы

Система управления проектами предназначена для комплексного управления корпоративными проектами с возможностью повторного использования планирования, автоматического подбора сотрудников и запуска бизнес-процессов по шаблонам. Система обеспечивает прозрачное отслеживание выполнения задач, управление командой и аналитику производительности.

## 2. Основные экраны

### Экран 1: Главная панель (Dashboard)

**Назначение:** Обзор активных проектов и задач пользователя

**Элементы интерфейса:**

- Заголовок "Система управления проектами"
- Навигационное меню (Проекты, Рабочая область, Команда, Аналитика)
- Карточки активных проектов с статусом и прогрессом
- Список назначенных задач с дедлайнами
- Кнопка "Создать протокол" (основное действие)

**Интерактивные элементы:**

- Кнопка "Создать протокол" → переход к Экрану 2

### Экран 2: Создание протокола

**Назначение:** Инициация нового бизнес-процесса или проекта

**Элементы интерфейса:**

- Форма с полями:
  - Название протокола
  - Описание
  - Тип (Проект/Бизнес-процесс)
  - Приоритет (Низкий/Средний/Высокий/Срочный)
  - Дедлайн
- Кнопки: "Отмена", "Далее"

**Интерактивные элементы:**

- Кнопка "Далее" → переход к Экрану 3
- Кнопка "Отмена" → возврат к Экрану 1

### Экран 3: Выбор шаблона

**Назначение:** Выбор шаблона планирования или бизнес-процесса

**Элементы интерфейса:**

- Список доступных шаблонов с описанием и метаданными
- Фильтры по домену и сложности
- Карточки шаблонов с:
  - Названием и версией
  - Описанием
  - Средней длительностью
  - Уровнем сложности
- Кнопки: "Назад", "Выбрать шаблон"

**Интерактивные элементы:**

- Выбор шаблона → выделение карточки
- Кнопка "Выбрать шаблон" → переход к Экрану 4
- Кнопка "Назад" → возврат к Экрану 2

### Экран 4: Назначение исполнителей

**Назначение:** Распределение задач между сотрудниками

**Элементы интерфейса:**

- Список задач из выбранного шаблона
- Для каждой задачи:
  - Название и описание
  - Требуемые компетенции
  - Предлагаемые исполнители с рейтингом соответствия
  - Кнопка назначения
- Панель с информацией о сотрудниках:
  - Фото и имя
  - Роль и уровень
  - Текущая загрузка
  - Рейтинг соответствия задаче

**Интерактивные элементы:**

- Выбор исполнителя для задачи → автоматическое назначение
- Кнопка "Назначить" → подтверждение назначения
- Кнопка "Завершить" → переход к Экрану 5

### Экран 5: Детали задачи

**Назначение:** Просмотр и обновление статуса конкретной задачи

**Элементы интерфейса:**

- Информация о задаче:
  - Название и описание
  - Исполнитель и статус
  - Дедлайн и SLA
  - Прогресс выполнения
- Форма отчетности:
  - Текстовое поле для комментариев
  - Загрузка файлов
  - Чек-лист выполнения
- Кнопки действий: "Начать выполнение", "Завершить", "Эскалировать"

**Интерактивные элементы:**

- Изменение статуса → обновление прогресса
- Заполнение формы → сохранение данных
- Кнопка "Завершить" → переход к Экрану 6

### Экран 6: Мониторинг исполнения

**Назначение:** Отслеживание прогресса проекта/процесса

**Элементы интерфейса:**

- Диаграмма Ганта с задачами и сроками
- Статус выполнения задач (To Do/In Progress/Review/Done)
- Прогресс-бар общего выполнения
- Список просроченных задач
- Уведомления об эскалациях
- Метрики: время выполнения, отклонения от плана

**Интерактивные элементы:**

- Клик на задачу → переход к Экрану 5
- Фильтры по статусу и исполнителю
- Кнопка "Вернуться к dashboard" → Экран 1

## 3. Сценарий пользователя

**Основной поток:**

1. Пользователь входит в систему → Dashboard (Экран 1)
2. Нажимает "Создать протокол" → форма создания (Экран 2)
3. Заполняет данные и переходит далее → выбор шаблона (Экран 3)
4. Выбирает подходящий шаблон → назначение исполнителей (Экран 4)
5. Распределяет задачи между сотрудниками → детали задачи (Экран 5)
6. Просматривает и обновляет статус задач → мониторинг (Экран 6)

**Альтернативные потоки:**

- Возврат на предыдущие экраны через кнопки "Назад"
- Прямой переход к мониторингу из dashboard
- Эскалация задач при просрочке SLA

## 4. Дизайн-система

**Цветовая палитра:**

- Основной: Slate (серый) для фона и текста
- Акцентный: Blue (#3B82F6) для кнопок и активных элементов
- Статусы: Green (успех), Red (риск), Amber (предупреждение)
- Нейтральный: Gray для границ и разделителей

**Типографика:**

- Заголовки: 18-24px, bold
- Основной текст: 14-16px, regular
- Метки и кнопки: 12-14px, medium

**Компоненты:**

- Карточки с закругленными углами (8px radius)
- Кнопки с hover эффектами
- Формы с валидацией
- Иконки для навигации и действий
